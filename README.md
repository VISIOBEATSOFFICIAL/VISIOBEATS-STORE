# 🎧 Visiobeats — Premium Beat Store

Welcome to **Visiobeats**, an independent online beat store built to deliver high-quality, original instrumentals for artists, creators, and brands.

## 🚀 About Me

I’m an 18-year-old music producer based in California, focused on creating modern, industry-style beats inspired by today’s top sounds. I’ve been actively building my presence through platforms like YouTube and TikTok, where I showcase my work and connect with artists.

My goal is simple:

> Create hard, unique beats and make them easily accessible for artists who want to stand out.

## 🌐 Project Overview

This website is a custom-built beat store that allows users to:

* 🎵 Preview beats directly in the browser
* 💳 Purchase beats securely using Stripe
* ⚡ Instantly access premium instrumentals

Built with:

* HTML, CSS, JavaScript (Frontend)
* Node.js + Express (Backend)
* Stripe API (Payments)

## 📦 Features

* Clean, modern UI
* Audio preview player
* Secure checkout system
* Scalable backend structure
* Ready for deployment (GitHub + hosting platforms)

## 🔐 Payments

All transactions are handled securely through Stripe. No sensitive payment information is stored on this server.
